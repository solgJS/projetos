objetos e ui

existe apenas um ui que seria a que o usuário ve seria os botão que fica fixo,mesmo se a camer se mecher os objetos são o mundo player,chão,fundo,bots,iten,etc.

•objetos

-var object = new object(): cria um objeto dentro aceita: x,y,tamanhoX,tamanhoY,cor/imagem,fisica(false/true) -var text = new text(): cria um texto aceita: x,y,tamanho,string,adicionar a um parente(objeto,ui) -var button = new button(): cria um botão ele será automaticamente adicionado a ui,aceita: x,y,tamanhoX,tamanhoY,texto/caminho da imagem,tamanho do texto,uma função para ser executado quando for tocado -var input = new input(): cria um entrada para o usuário responder aceita: x,y,tamanhoX,tamanhoY,valor inicial/ou só null,adicionar a um parente

funções

-objeto.camera() a camera era seguir o objeto se ele se mover -objeto.die() era definir a variável do objeto para "" e era apagar visual mente -objeto.step() fara o player andar aceita: direção x/y,velocidade,por quanto tempo -object-rotação() define a rotação aceita: rotação

propiedade

pra acesar alguma propriedade de um objeto você usa objeto.propiedade por exemplo: player.x = 0 alguamas propriedadee x,y,sizeX,sizeY,paddin,border,color
